# Encoder-Decoder Transformer from Scratch

A fully self-implemented Encoder-Decoder Transformer model trained on a German-to-English translation task.

## Report

The full project report is available at [Report.pdf](Report.pdf).


### Training

To run the training script:

```bash
poetry run python train_script.py
```

### Inference

To run the inference script:

```bash
poetry run python inference_script.py
```
